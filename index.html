Administração de Servidores

Gerenciando Comandos

Comandos Básicos

  • pwd - Comando para mostrar o diretório atual.

  • ls - Comando para listar os arquivos do diretório.

  • ls -la - Mostrar os arquivos detalhados.

  • date - Mostrar a data.

  • mk dir - Criar um novo diretório.

  • touch - Criar um novo arquivo.

  • vim nomedoarquivo - Abrir o arquivo.

  • kat nomedoarquivo - Zipar o arquivo sem abrir.

  • rm nomedoarquivo - Comando para deletar arquivo ou diretório.

Comandos de Inserção

  • i - Inserir conteúdo antes do cursor.

  • a - Inserir conteúdo depois do cursor.

  • I - Ir para o início da linha.

  • A - Ir para o final da linha.

  • o - Quebra a linha para baixo.

  • O - Quebra a linha para cima.

Comandos Normais

  • h - Move o cursor para esquerda.

  • l - Move o cursor para a direita.

  • j - Move o cursor para baixo.

  • k - Move o cursor para cima.

  • gg - Ir para o início do arquivo.

  • G - Ir para o final do arquivo.

  • 0 - Ir para o início da linha.

  • Shift + $ - Ir para o final da linha.

  • w - Pular palavra para frente.

  • b - Pular palavra para trás.

Copiar, colar e deletar

  • yy- Copiar linha.

  • p - Colar a linha copiada.

  • yw - Copiar palavra.

  • dd - Apagar uma linha.

  • dw - Apagar uma palavra.

  • x - Deletar um caractere específico.

  • u - Desfazer.

  • Ctrl + r - Refazer.

Busca e substituição

  • / - Comando para buscar determinada palavra (a palavra deve ser digitada depois da barra.)

  • n - Caso exista mais de uma palavra na busca, digita (n) para pular para a próxima ocorrência.

  • N - Voltar para a ocerrência anterior.

  • :s/palavra/novapalavra - Procurar palavras abaixo e substituir.

  • %s/palavraantiga/palavranova - Comando para procurar e subtituir palavra em todo o arquivo.
